## 暴露服务端口

```bash
itux@k8s01:~> sudo kubectl edit -n monitoring svc prometheus-k8s
spec:
  clusterIP: 10.97.156.163
  ports:
  - name: web
    port: 9090
    protocol: TCP
    targetPort: web
    nodePort: 30090
  selector:
    app: prometheus
    prometheus: k8s
  sessionAffinity: None
  type: NodePort

itux@k8s01:~> sudo kubectl edit -n monitoring svc alertmanager-main
spec:
  clusterIP: 10.102.61.156
  ports:
  - name: web
    port: 9093
    protocol: TCP
    targetPort: web
    nodePort: 30091
  selector:
    alertmanager: main
    app: alertmanager
  sessionAffinity: None
  type: NodePort

itux@k8s01:~> sudo kubectl edit -n monitoring svc grafana
spec:
  clusterIP: 10.107.108.89
  ports:
  - name: http
    port: 3000
    protocol: TCP
    targetPort: http
    nodePort: 30092
  selector:
    app: grafana
  sessionAffinity: None
  type: NodePort
```

* 修改svc，将prometheus、alertmanager、grafana通过nodeport暴露方便访问；
